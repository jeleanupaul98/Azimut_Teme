Tag-uri HTML
Tag-uri Descriere / Exemple
< p > ... < / p > Textul unui paragraf
< h1 > ... < / h6 > Titluri si subtitluri. "h1" este cel mai mare iar "h6" este cel mai mic. Are tag de inchidere.
< ol > ... < /ol >
< li > ... < /li >
< ul > ... < / ul >		 Liste ordonate, liste neordonate, si elementele lor.
ex1:
< ol type="A" >
< li > Aiud < /li >
< li > Teius < /li >
< /ol >
< b >... < /b > Text ingrosat. Are tag de inchidere. Ingroasa textul cuprins intre etichete.
Exemplu:
< p >Acesta este un text < b >ingrosat< /b >.< /p >
< a ... > ... < /a > Anchor tag. Este folosit pentru a crea hyperlink-uri folosite in navigarea dintre pagini web. Are tag de inchidere.
Foloseste atribute precum href pentru a seta destinatia, sau target pentru a descrie cum se va deschide pagina/documentul specificat.
Exemplu:
< p > < a href = " https://ro.wikipedia.org/Franta " > Franta < /a > este a doua putere economica europeana si a cincea putere mondiala. < /p >
< table >< /table >
< th >< /th >
< tr > < /tr >
< td > < / td >
< caption >< /caption >		 Tabel. Are tag de inchidere.
TH reprezinta capetele de tabela.
TR reprezinta randurile tabelei.
TD reprezinta coloanele tabelei.
Caption reprezinta titlul tabelei.
Exemplu:
< table >
< caption > Inventar fructe < /caption >
< tr >
< th > Fruct < /th >
< th > Cantitate < /th >
< /tr >
< tr >
< td > Mere < /td >
< td > 70 < /td >
< /tr >
< tr >
< td > Banane < /td >
< td > 30 < /td >
< /tr >
< /table >
< abbr >...< /abbr > Abreviere. Are tag de inchidere.
Se foloseste pentru a marca abrevierile. Foloseste atributul "title" pentru a marca semnificatia abrevierii. Are ca efect afisarea semnificatiei abrevierii cand se face hover cu cursorul peste cuvantul abreviat.
Exemplu:
< abbr title= "World Health Organization" >WHO< /abbr > a fost fondata in anul 1948.
< audio ...> ... < /audio >
< source ... >		 Tag-ul "audio" este folosit pentru inserarea si redarea fisierelor audio in pagina. Are tag de inchidere.
Tag-ul copil "source" este folosit pentru a indica sursa, locatia fisierului, si are atributele "src" (pentru indicarea sursei) si "type" (pentru indicarea tipului de fisier)
Cand sunt oferite mai multe surse pentru acelasi tag audio, browser-ul va afisa in pagina prima sursa gasita din lista.
Exemplu:
< audio controls >
< source src="/muzica/audio.ogg" type="audio/ogg">
< source src="muzica/audio.mp3" type="audio/mpeg">
< /audio >
< br > Line break, nu are tag de inchidere.
Se foloseste pentru a trece la o linie noua. Cand browser-ul citeste acest tag, stie ca trebuie sa afiseze urmatoarea linie/urmatorul element cu un spatiu mai jos.
Exemplu:
Aceasta este < br > o linie noua.
< button ... >< /button > Eticheta de buton. Are tag de inchidere.
Este folosit pentru crearea unui buton.
Foloseste atribute precum "type" pentru a indica tipul de buton (button/reset/submit), tipul descrie comportamentul pe care il va avea butonul.
Ii poate fi asociata o valoare prin atributul "value", sau un nume prin atributul "name".
Exemplu:
< button type="button" >Apasa aici!< /button >
sau
< button type="reset"> Apasa aici pentru a reseta formularul!< /button >
